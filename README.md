# 500 Poems 

## What's here
- `index.html` — the entire viewer (HTML/CSS/JS + all page text embedded). No build step, no dependencies.

## Hosting on GitHub Pages
1. Create a new GitHub repo named `500-poems` (or `500 Poems`).
2. Upload `index.html` to the repo root (GitHub web UI: **Add file → Upload files**).
3. Go to **Settings → Pages**, set source to the `main` branch, root folder.
4. GitHub will publish it at `https://<your-username>.github.io/500-poems/`.

## Local preview
Just open `index.html` in any browser — no server required.
